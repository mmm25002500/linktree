# TershiXia LinkTree(夏特稀連結樹)
A linktree(lntr) with React.

## Installation

```bash
# install necessary pkg listed in package.json
yarn
```

## Run
```bash
# run daemend in bg
yarn dev
```

## Deploy
1. Edit ```script``` in ```package.json```.

```bash
# deploy by this cmd
yarn deploy
```
